# shalinraj_25scs1003002739_IILM-GN
A leaf disease recognition system that uses image processing and machine learning to detect and classify plant diseases from leaf images. It helps farmers and researchers identify issues early, enabling timely treatment and improving crop health and productivity.

---

## 🚀 Features

- 🌱 Leaf disease classification using images  
- 🧠 Transfer learning with ResNet18  
- ⚡ Fast and accurate predictions  
- 🖥️ Simple CLI-based prediction system  
- 🔌 Easy to extend for new datasets  

---

## 🧠 Model Details

- Architecture: **ResNet18 (Pre-trained)**
- Framework: **PyTorch**
- Training: Transfer Learning
- Output: Multi-class classification
